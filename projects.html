SkillSync – AI-Powered Job Title Recommendation

SkillSync is an AI-based career support web app designed to recommend personalized job titles and evaluate skill readiness based on user input. It uses Natural Language Processing (NLP) and machine learning to compare user skills with a database of over 1,000 job roles and suggests career paths with skill gap analysis.

  • Built using Flask, Pandas, and Scikit-learn.
  • Implements cosine similarity for job recommendation engine.
  • Integrated with a responsive HTML/CSS/Bootstrap frontend.
  • Offers resume feedback and mock interview preparation tools.
  • Achieved 87% accuracy in matching profiles to relevant job roles.

🚀 SkillSync – AI-Powered Skill Gap Analyzer

SkillSync is a smart web app that identifies skill gaps by comparing user skills with job descriptions using AI. It helps learners and professionals find what they’re missing and provides a personalized upskilling roadmap.

  • ✅ Skill-to-job matching with NLP
  • 📉 Highlights missing or weak skills
  • 📚 Personalized upskilling recommendations
  • 🔁 Keeps users aligned with job market trends

Tech Stack: Python, Flask, TensorFlow, Scikit-learn, HTML5, Tailwind CSS

🎤 AI-Powered Interview Analysis Web App

An interactive web application simulating real-time technical interviews using AI and NLP. It generates role-specific questions, records user answers, transcribes speech to text, and evaluates responses with intelligent feedback.

Key Features:

  • 🧠 AI-generated interview questions using Google Gemini
  • 🎙️ Speech-to-text answer capture via Google SpeechRecognition
  • ✅ Real-time validation of answers with detailed feedback
  • 📊 Tracks user performance and generates Excel reports
  • 🗃️ NLP-based filler word detection and fluency scoring

Tech Stack: Python, Flask, Google Gemini API, NLTK, Pandas, Google SpeechRecognition, Excel (xlsx), HTML5

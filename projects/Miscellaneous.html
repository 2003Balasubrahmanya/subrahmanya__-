📥 YouTube_download – Video Downloader UI

YouTube_download is a front-end interface designed for downloading YouTube videos. It provides a clean and intuitive layout where users can input a video URL and select formats. Intended to be used with a backend like pytube or youtube-dl for full functionality.

🔹 Key Features

  • Simple UI: Clean input field for video URL with clear instructions.
  • Download Button: Ready-to-hook interface for triggering downloads via backend.
  • Extensible Design: Easily integrates with backend APIs to handle actual download logic.
  • Responsive Layout: Mobile-friendly design with adaptive layout.

Tech Used: HTML, CSS, JavaScript
